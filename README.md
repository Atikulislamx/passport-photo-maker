# Passport Photo Maker

A modern web application that automatically converts any portrait photo into a passport-size photo directly in the browser.

## Features

- Upload portrait photo
- Automatic face detection
- Smart face centering
- Background removal
- Passport size formatting (35mm x 45mm)
- Brightness and contrast adjustment
- Preview before and after editing
- Download passport photo
- Generate 4x printable photo sheet

## Supported Formats

- JPG
- PNG

Maximum upload size: **5MB**

## How It Works

1. Upload a portrait image.
2. The system detects the face automatically.
3. The image is cropped and aligned according to passport photo rules.
4. Background is removed or replaced.
5. Image is resized to passport dimensions.
6. Download the final passport photo.
